Documentation

To start a scan, go to the main page and click the Start scanning button. Select the date of the scan (the current date is recommended) and select a group of employees, if the scan is not performed for all employees.
Press the start button.
You will see a progress bar that will show the progress of the scan, upon completion the updated data will be available for use.
To begin with, you need to refer to the grouped list. People who are in the high-risk zone must be examined in more detail, as well as people whose condition has worsened. For more detailed research, you need to go to the employee's personal page. There you can see graphs based on indicators, which will help to ascertain the state of the employee.
On the employee's personal page, you can see the change in 5 indicators: the number of tweets written per month, the percentage of normal, stressful, lonely tweets and in addition indicator of burn-out (sum of stressed and lonely percentages). If there are any sudden changes in the indicators, this is a reason to pay attention to this employee.
If a person is at risk, first you need to talk to the employee's manager and ask if he has noticed any changes in his work or communication with the employee. If the changes were in a negative direction, it is necessary to contact the employee himself and inquire about his condition, indicate that the company provides the services of a psychologist free of charge and offer to take a course.
